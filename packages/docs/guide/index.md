# Огляд системи

**Helpdesk** — це веб-застосунок для автоматизації сервісної підтримки, управління мережевою інфраструктурою та обліку звернень користувачів.

## Архітектура

Система побудована як монорепозиторій з трьома пакетами:

| Пакет           | Технологія          | Призначення                  |
| --------------- | ------------------- | ---------------------------- |
| `packages/api`  | NestJS 11 + MongoDB | GraphQL API серверна частина |
| `packages/app`  | Vue 3 + Vite        | Клієнтський веб-застосунок   |
| `packages/docs` | VitePress           | Документація                 |

## Основні можливості

- **Заявки (Requests)** — реєстрація, обробка та закриття заявок на технічну підтримку
- **IP-адреси** — облік та управління мережевими адресами
- **Поштові скриньки** — управління корпоративними email-обліковими записами
- **Мережеві канали** — моніторинг каналів зв'язку між локаціями
- **SysInspector** — збір та перегляд інформації про ПК
- **Звіти** — конфігуровані звіти з вивантаженням у Excel
- **Статистика** — аналітичні дашборди по мережі, пошті, заявках та ПК
- **Довідники** — організації, підрозділи, відділи, посади, локації, пристрої
- **Аудит** — системний журнал дій користувачів
- **Сповіщення** — система персональних сповіщень

## Технологічний стек

### Серверна частина (API)

- **NestJS 11** — основний фреймворк
- **GraphQL** (Apollo Server 5) — транспортний протокол, code-first підхід
- **MongoDB + Mongoose 9** — база даних
- **JWT** (access + refresh токени) — автентифікація
- **Passport.js** — стратегії автентифікації
- **DataLoader** — батчинг запитів, вирішення проблеми N+1

### Клієнтська частина (App)

- **Vue 3** — Composition API
- **Vite** — збірка
- **Pinia** — управління станом
- **Vue Router 4** — маршрутизація
- **Apollo Client 3** — GraphQL клієнт
- **PrimeVue** — UI компоненти
- **MDI Icons** — іконки

```
│   │       ├── views/        # Сторінки
│   │       ├── components/   # Компоненти
│   │       ├── stores/       # Pinia stores
│   │       ├── graphql/      # GQL запити
│   │       └── ...

```
