# ПК SysInspector

Специализированный модуль для сбора и анализа аппаратных и программных характеристик компьютеров в сети.

## Собираемые данные

Сервис аккумулирует информацию по следующим категориям:

- **Hardware**: Процессор (CPU), объем оперативной памяти (RAM), модель материнской платы.
- **Storage**: Список дисковых накопителей и их объем.
- **Software**: Версия операционной системы, установленные обновления и специфическое ПО.
- **Сетевые интерфейсы**: Активные сетевые карты и их настройки.

## Применение

Данные используются ИТ-отделом для:

- Планирования апгрейдов парка ПК.
- Выявления компьютеров с устаревшим ПО.
- Быстрого поиска характеристик устройства при удаленной поддержке.
