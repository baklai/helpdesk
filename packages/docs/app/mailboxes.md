# Поштові скриньки (Mailboxes)

Модуль управления корпоративными почтовыми аккаунтами.

## Функционал

- **Реестр адресов**: Единая база всех корпоративных email-адресов.
- **Привязка к сотрудникам**: Каждый ящик связан с конкретным пользователем в системе.
- **Управление квотами**: Мониторинг лимитов дискового пространства почтовых ящиков (если поддерживается сервером).

## Интеграция

Данные о почтовых ящиках используются в модуле "Requests" для автоматического уведомления пользователей о статусе их заявок.
