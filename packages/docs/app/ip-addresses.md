# Мережева інформація (IP Addresses)

Этот модуль предназначен для инвентаризации и мониторинга IP-адресов в локальной сети предприятия.

## Основной функционал

- **Визуализация статусов**: Отображение доступности хостов (Online/Offline) в реальном времени.
- **Фильтрация**: Поиск по IP, MAC-адресу, имени пользователя или подразделению.
- **Инструменты хоста**: Генерация файлов для быстрого управления устройствами.

## Технические детали

Данные подтягиваются через запрос `FIND_ALL_IPADDRESSES`. Каждая запись содержит:

- `ip`: IP-адрес устройства.
- `mac`: Физический адрес.
- `department`: Объект подразделения (название, описание).
- `location`: Географическое или структурное расположение.

## Инструменты администратора (useHostTools)

В интерфейсе реализованы кнопки для генерации скриптов:

1. **PING**: Скачивает `.cmd` файл для проверки отклика.
2. **RDP**: Файл для подключения через удаленный рабочий стол Windows.
3. **VNC**: Файл конфигурации для удаленного управления.

> [!TIP]
> Генерация происходит на стороне клиента путем преобразования Base64 ответа от сервера в Blob-объект.
